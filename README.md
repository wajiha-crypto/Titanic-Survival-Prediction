# Titanic Survival Prediction

## Project Description
This project was developed as part of a Data Science internship at Arch Technologies. The system predicts whether a Titanic passenger would have survived the disaster using passenger information such as ticket class, gender, age, number of siblings/spouses aboard, number of parents/children aboard, fare, and port of embarkation.

## Objectives
- Apply machine learning concepts to solve a real-world classification problem.
- Analyze and preprocess real-world, messy data.
- Build a survival prediction model.
- Create an interactive interface for users to test predictions.

## Project Workflow
1. Data loading and preprocessing
2. Handling missing values
3. Encoding categorical variables
4. Data visualization and correlation analysis
5. Model training using Logistic Regression
6. Model evaluation
7. Survival prediction through a Gradio interface

## Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Gradio

## Model Performance
The model was evaluated using:
- Accuracy Score (81%)
- Confusion Matrix
- Precision, Recall, F1-Score

## Key Insights
- Gender was the strongest predictor of survival, consistent with the historical "women and children first" policy.
- Passenger class also played a major role — 1st class passengers had significantly higher survival rates than 3rd class.

## User Interface
A Gradio-based interface allows users to enter passenger details and receive an instant survival prediction with probability.
